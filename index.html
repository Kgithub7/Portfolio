Here are some of my projects

  • Skinstric A.I.

    HTML, Tailwind CSS, JavaScript, React, Vite

    Skinstric is a modern, responsive web application, recreating a portion of the official Skinstric A.I. site based on provided Figma designs. Designed for analyzing skin conditions from images, it allows users to either upload an image from their gallery or capture a new one using their device's camera. The application then provides predictions of the user's demographic, simulating the AI-driven skin analysis process of the original platform.

  • Netflix

    HTML, Tailwind CSS, TypeScript, React, Vite, Firebase

    This project is a functional clone of the Netflix user interface. It leverages the TMDB (The Movie Database) API to fetch and display real movie and TV show data, providing users with a dynamic browsing experience. It also includes user authentication powered by Firebase and a fully responsive visual presentation across devices using custom CSS.

  • ShowScout

    HTML, CSS, JavaScript, React, Vite

    ShowScout is a sleek and responsive TV show search engine. It connects to a live API to help users discover new shows and instantly find relevant information, including summaries and official links, all within a clean and intuitive interface.

  • Library

    HTML, CSS, JavaScript, React, Vite

    Library is a fully responsive e-commerce storefront for an online bookstore. It provides a complete shopping experience, from browsing featured and discounted books on the homepage to sorting the full catalog and managing a functional shopping cart.
